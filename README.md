# React-Cursos
Repositório com os códigos do curso React do Zero a Maestria.<br>

Todos os códigos foram realizados com: <br><br> ![TypeScript](https://img.shields.io/badge/typescript-%23007ACC.svg?style=for-the-badge&logo=typescript&logoColor=white) ![React](https://img.shields.io/badge/react-%2320232a.svg?style=for-the-badge&logo=react&logoColor=%2361DAFB) <br>

[ReactGram](https://github.com/EduardoAVS/React-Cursos/tree/main/reactgram): Rede social para compartilhamento de fotos, onde os usuários podem fazer upload de imagens, visualizar postagens de outros usuários e interagir com curtidas e comentários. <br><br>![Redux](https://img.shields.io/badge/redux-%23593d88.svg?style=for-the-badge&logo=redux&logoColor=white) ![Express.js](https://img.shields.io/badge/express.js-%23404d59.svg?style=for-the-badge&logo=express&logoColor=%2361DAFB)  ![MongoDB](https://img.shields.io/badge/MongoDB-%234ea94b.svg?style=for-the-badge&logo=mongodb&logoColor=white) ![NodeJS](https://img.shields.io/badge/node.js-6DA55F?style=for-the-badge&logo=node.js&logoColor=white)      <br>

[SecretWord](https://github.com/EduardoAVS/React-Cursos/tree/main/SecretWord): Criei um jogo interativo onde os jogadores devem adivinhar palavras com base em dicas e letras reveladas gradualmente. O jogo inclui um sistema de pontuação e limite de tentativas. <br>

[to-do list](https://github.com/EduardoAVS/React-Cursos/tree/main/todo-list): Desenvolvi uma aplicação de lista de tarefas (To-Do) que permite aos usuários adicionar, editar, marcar como concluído e excluir tarefas. <br>
